# shrink
Get any URL Shortened in your terminal

<h1>How to use it</h1>

```
sh shrink.sh "http://www.google.com"

```
In Terminal Results :

```
✔ Original URL :http://www.google.com
✔ Short URL    :"https://3o2.co/S4"
✔ QR Code      :"https://3o2.co/S4.qrcode"

```
